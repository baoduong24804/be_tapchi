API Documentation

POST
Check Roles
/api/user/checkRoles
Dữ liệu:
  • token: Token của người dùng
POST
Lấy Thông Tin Người Dùng
/api/user/get/userDetail
Dữ liệu:
  • token: Token của người dùng
POST
Đăng Nhập
/api/user/login
Dữ liệu:
  • username: Tên đăng nhập hoặc email
  • password: Mật khẩu
POST
Đăng Ký
/api/user/register
Dữ liệu:
  • username: Tên đăng nhập
  • password: Mật khẩu
  • email: Email người dùng
  • sdt: Số điện thoại
  • hovaten: Họ và tên
  • url: Ảnh đại diện URL
POST
Đổi Mật Khẩu
/api/user/changepassword
Dữ liệu:
  • username: Tên đăng nhập
  • password: Mật khẩu cũ
  • newpassword: Mật khẩu mới
POST
Quên Mật Khẩu
/api/user/forgot
Dữ liệu:
  • email: Email để gửi yêu cầu đặt lại mật khẩu
POST
Đặt Lại Mật Khẩu
/api/user/reset
Dữ liệu:
  • token: Token xác thực đặt lại mật khẩu
  • password: Mật khẩu mới
POST
Upload File
/api/files/upload
Dữ liệu:
  • files: Danh sách tệp tin cần tải lên (required)
Phản hồi:
  • Status 200: {"message": "File uploaded successfully: [location] file1,file2,..."}
  • Status 400: {"message": "No files provided"}
DELETE
Delete File
/api/files/delete/{fileName}
Dữ liệu:
  • fileName: Tên của tệp tin cần xóa (required)
Phản hồi:
  • Status 200: {"message": "File deleted successfully: {fileName}"}
  • Status 500: {"message": "Failed to delete file: {fileName}"}
DELETE
Delete All Files
/api/files/delete-all
Phản hồi:
  • Status 200: {"message": "All files deleted successfully."}
  • Status 500: {"message": "Failed to delete all files."}
GET
Download File
/api/files/download/{fileName}
Dữ liệu:
  • fileName: Tên của tệp tin cần tải xuống (required)
Phản hồi:
  • Status 200: Tệp tin được tải xuống (attachment).
  • Status 404: {"message": "File not found."}
GET
View Image
/api/files/view/{fileName}
Dữ liệu:
  • fileName: Tên của tệp hình ảnh cần xem (required)
Phản hồi:
  • Status 200: Hình ảnh được hiển thị.
  • Status 404: {"message": "File not found."}
GET
Get All Files
/api/files/view/all
Phản hồi:
  • Status 200: ["file1", "file2", ...]
  • Status 500: {"message": "Error loading file list."}
